# 봉선 꿰뚫영어 · 화정 더오름영어 - 정적 홈페이지 + 블로그 키트

## 1) 홈페이지(정적 사이트) 미리보기
- index.html 를 브라우저로 열면 됩니다.
- 폴더 구조:
  - /index.html
  - /pages/** (각 메뉴 페이지)
  - /assets/css/style.css (디자인)
  - /assets/js/main.js (모바일 메뉴 토글)
  - /assets/images/ (이미지 자리)

## 2) 이미지 적용 방법
- assets/images/ 폴더에 원장님 제공 이미지 파일을 넣고
- 각 페이지에서 <img> 태그로 연결하면 됩니다.
- 현재는 ‘문구/구조’ 위주로 제작되어 있습니다.

## 3) 무료로 링크(주소) 만들기: GitHub Pages 추천
1. GitHub 가입
2. 새 Repository 생성 (예: kke-ddo-eom)
3. 이 폴더 내용을 전부 업로드
4. Settings → Pages → Build and deployment
   - Source: Deploy from a branch
   - Branch: main / (root)
5. 저장하면 아래 형태로 주소가 생성됩니다:
   https://<아이디>.github.io/<레포이름>/

## 4) 블로그 글 올리기(네이버)
- blog-kit/markdown: 원고(원장님 수정용)
- blog-kit/naver-html: 네이버 에디터에 ‘HTML’ 모드로 붙여넣기용
  (스마트에디터 ONE은 ‘HTML 직접 붙여넣기’가 제한될 수 있어,
   일반 붙여넣기 후 굵게/줄바꿈만 약간 정리해도 됩니다.)

## 5) 꼭 채우면 좋은 정보(페이지 빈칸)
- 봉선/화정 주소, 지도 링크(네이버지도/카카오맵), 주차 안내
- 수업 시간표(요일/시간), 수강료(표로)
- 성적표/후기 사례(개인정보 블라인드) + 합격 사례
